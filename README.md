# InsightStream: Self-Evolving Research AI
Adaptive engineering research that learns from your expertise.

🚀 The Problem
Engineering research is plagued by "Information Entropy." Standard LLMs provide generic, static responses. InsightStream evolves its search policy using Agentic Workflows and RLHF to prioritize high-credibility technical sources.

🛠 Tech Stack
Frontend: React.js, Tailwind CSS, Vite

Backend: Python (FastAPI), LangChain

Intelligence: Gemini API

Memory: ChromaDB (Vector Store)

📂 Project Structure
/frontend: React dashboard and UI components.

/backend: Python agents, scrapers, and API logic.

/docs: Research papers, prompts, and presentation assets.

⚙️ Setup Instructions
Frontend:

cd frontend

npm install

npm run dev

Backend:

cd backend

pip install -r requirements.txt

python main.py
